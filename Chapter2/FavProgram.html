Miles Per Gallon Calculator

My favorite program that I have done so far was called Miles Per Gallon Calculator.

This program asks the user about the last ten trips they have taken.

This could be as simple as going to the gas station, or going across the whole country.

The user has to input their starting gallons, ending gallons and how many miles they traveled.

It then calculates the MPG for that trip and up to ten trips.
